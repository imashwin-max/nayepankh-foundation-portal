# 🕊️ NayePankh Foundation — AI-Powered NGO Portal

A fully functional, AI-powered web portal built for NayePankh Foundation 
as part of the Vibe Coder Internship (Option 2).

## 🌟 Features

### 🤖 AI Chatbot — "Pankhi"
- Powered by Claude AI (Anthropic API)
- Trained on NayePankh's programs, internships, donations & volunteer info
- 7 quick-question shortcuts for common queries
- Full conversation memory within session
- Available 24/7 with warm, Hindi-friendly responses

### 🏠 Home Page
- Hero section with live impact stats (2L+ people helped, 30K+ interns)
- 6 program cards covering all NGO initiatives

### 📝 Intern Registration Form
- Complete application form with validation
- 9 internship domain options
- Perks & benefits sidebar

### 📊 Impact Dashboard
- Animated counters (2L+ people, 8.5L+ meals, 30K+ interns)
- Animated bar chart by program type
- Donut chart for intern domain breakdown
- Recent applications table with status badges

### 💙 Donation Page
- Preset amounts with real impact descriptions (₹100 = 2 meals)
- Custom amount input + PAN field for 80G tax certificate
- Monthly fundraising progress bar

## 🛠️ Built With
- HTML5, CSS3, JavaScript (Vanilla)
- Claude AI API (Anthropic) — claude-sonnet-4-6
- Google Fonts (Playfair Display, Inter, Space Mono)
- No frameworks, no dependencies — single HTML file
